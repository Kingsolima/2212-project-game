Staying Alive - Typing Game
===========================

This repository contains the source code and assets for **Staying Alive**, a Java Swing educational typing game built for COMPSCI 2212.


## Structure

- `docs/` – planning, design, and architecture documents.
- `assets/` – images, audio, and fonts used in the game.
- `data/` – JSON files for player and master data (local storage).
- `lib/` – third-party JARs such as Jackson and TinySound.
- `src/main/java/` – Java source tree for the main application.
- `src/test/java/` – JUnit tests for non-GUI logic.
- `test/` – any additional manual or integration test resources.
